# Description

Implementing a smart tracker that captures player movements in a gameplay and provides game statistics

# Features

Features include -

1. precise player, referee and ball tracking throughout the game
2. special trackers on players controlling the ball at a given time
3. team possession stats on how much each team controlled the ball

Custom YOLOv5 training using Roboflow :

1. enhance image detection and differentiate referee and players from other non-playing people in the frame
2. continous ball detection and tracking throughout the video
